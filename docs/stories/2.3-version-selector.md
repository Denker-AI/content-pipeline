# Story 2.3: Version Selector

## Goal

Add a version dropdown/tabs for content that has multiple versions (draft v1, v2, ... final). Let user switch between versions in the preview.

## Dependencies

- Story 2.2 (preview renderer)

## Files to Create/Edit

```
src/renderer/components/version-selector.tsx  # Version tab bar / dropdown
src/renderer/components/preview-pane.tsx      # Update: integrate version selector
src/renderer/hooks/use-content.ts            # Update: version detection
```

## Implementation

- Scan newsletter directory for `drafts/v*.html` files
- Show tab bar above preview: `v1 | v2 | ... | final`
- "final" = `email.html` (highlighted/bold)
- Clicking a version loads it in the preview
- Auto-detect when new versions are created (via file watcher)

## Acceptance Criteria

- [ ] For newsletters with drafts, version tabs appear above preview
- [ ] Clicking a version tab switches the preview content
- [ ] Final version (email.html) is visually distinguished
- [ ] New draft versions appear automatically when created
- [ ] Content without versions shows no version bar
