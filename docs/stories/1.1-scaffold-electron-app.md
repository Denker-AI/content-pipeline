# Story 1.1: Scaffold Electron App

## Goal

Create a minimal Electron app that opens a window. Set up the entire build toolchain: Vite for renderer bundling, TypeScript, Tailwind CSS, React, electron-builder.

## Files to Create

```
package.json
tsconfig.json
vite.config.ts
tailwind.config.ts
electron-builder.yml
src/main/index.ts           # Electron main process entry
src/main/window.ts          # Window creation, menu
src/renderer/index.html     # HTML entry point
src/renderer/main.tsx       # React entry point
src/renderer/App.tsx        # Root component (placeholder)
src/renderer/styles/globals.css  # Tailwind directives
src/shared/types.ts         # Shared IPC types (empty for now)
.gitignore
.eslintrc.json
.prettierrc
```

## Tech Decisions

- **Package manager**: bun
- **Bundler**: Vite with `vite-plugin-electron` (handles main + renderer)
- **React**: 19.x
- **Tailwind**: 4.x
- **TypeScript**: strict mode
- **electron-builder**: for packaging (configured but not tested yet)

## Acceptance Criteria

- [ ] `bun install` succeeds
- [ ] `bun dev` opens an Electron window with "Content Pipeline" title
- [ ] Window shows a React component with "Hello Content Pipeline" text
- [ ] Tailwind classes work (e.g., `bg-zinc-900 text-white`)
- [ ] `bun build` produces a working build
- [ ] TypeScript compiles without errors
- [ ] ESLint passes
