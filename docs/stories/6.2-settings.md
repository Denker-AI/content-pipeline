# Story 6.2: Settings Panel

## Goal

Settings accessible from the app menu. Configure: app URL, API keys, persona, theme.

## Dependencies

- Story 1.1 (scaffold)

## Files to Create/Edit

```
src/main/settings.ts                        # Read/write settings file
src/main/ipc.ts                             # Update: settings IPC channels
src/renderer/components/settings-panel.tsx   # Settings form
src/shared/types.ts                         # Update: settings types
```

## Settings

| Setting | Type | Default | Storage |
|---------|------|---------|---------|
| App URL | string | `http://localhost:3000` | per-user |
| Auth cookies | object | `{}` | per-user |
| LinkedIn access token | string | — | per-user |
| Resend API key | string | — | per-user |
| Blog webhook URL | string | — | per-user |
| Persona (company, product, tone, audience) | object | — | per-project |
| Theme | `light` / `dark` | `dark` | per-user |

## Storage

- Per-user settings: `~/.content-pipeline/settings.json`
- Per-project settings: `content-pipeline.json` in project root (committed to git)

## Acceptance Criteria

- [ ] Settings accessible from app menu (Cmd+,)
- [ ] All settings listed above are configurable
- [ ] Settings persist across app restarts
- [ ] Per-user secrets (API keys) are NOT in project directory
- [ ] Per-project config (persona) IS in project directory
- [ ] Theme toggle works (light/dark)
