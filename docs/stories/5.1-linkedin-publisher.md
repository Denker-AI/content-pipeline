## Implementation

- Read `post-text.md` for post body
- Read `carousel-images/*.png` for images
- LinkedIn API: OAuth2 for auth, Share API for posting
- Supports: text-only, single image, carousel (document upload)
- After posting: update `post.json` with `linkedin_post_id` and `status: published`

## Acceptance Criteria

- [ ] "Publish to LinkedIn" button appears for LinkedIn content
- [ ] Shows post text preview and attached images
- [ ] Publishes text post via LinkedIn API
- [ ] Publishes carousel (multi-image) via LinkedIn document upload
- [ ] Updates post.json with post ID and published status
- [ ] Error handling: show clear error if auth fails

## Notes

- LinkedIn OAuth2 flow: separate Settings page handles token acquisition
- For MVP, require the access token to be set in settings (no in-app OAuth flow)
- Carousel upload requires PDF conversion of images (LinkedIn API quirk)
